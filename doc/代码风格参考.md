# 插件代码风格参考

这只是一份代码风格建议，不强制要求。

## Javascript

### 驼峰命名

使用 qiuShiGou 而不是 qiu\_shi\_gou

### JS-Doc

如：

```javascript
/**
* 获取物品信息
*
* @param {Object} args 参数
* @param {Function} callback function to callback when success
* @param {Function} callback function to callback when fail
*/
```

## CSS

### 属性命名顺序

没什么特别要求，尽可能按照a-z排序
